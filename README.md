# escape-game

これは技術書店2018の「WebGLの毎秒60回書き換わるStateをどう扱うか」の章のために作成したサンプルゲームです。

## How to install

```
npm install --save @sawa-zen/escape-game
```

## Getting Started

Install packages

```
npm install
```

### How to develop

```
npm start
```

### How to production build

```
npm build
```
